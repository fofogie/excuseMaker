# excuseMaker
So i got bored and made this very very simple application that gives you 8100 possible excuses for being late ranging from pretty funny to serious...

Again this is just a fun project that i did for a few minutes because i got nothing to do and i find it funny that some people come late to work/class/event with the most random reasons known to man..
So i took that joke and make a very simple program out of it and here is the result.
Features : 
1. A randomized sentence maker
2. Copy paste button that copies the sentence made into your clipboard

Enjoy making excuses :DD
